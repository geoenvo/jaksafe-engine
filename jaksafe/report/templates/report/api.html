Open Data Collection

Data from JakSAFE is available for public under an open license, and can be accessed by the link below:

jaksafe.bpbd.jakarta.go.id/report/auto_report_json/20160330/all

This API link will generate output every six hours (based on flood report update by BPBD), they are at 06.00, 12.00, 18.00, and 00.00. As example, data accessed at 06.00 is flood damage and loss estimation report for the latest 6 hours (00.00 - 06.00).
